# 说明

## 1. 工程文件使用方法

signal文件夹为工程文件夹，直接复制到自己的文件目录下就可以运行。

### 1.1 运行方法：

- 安装好quartus
- 双机.qpf文件打开工程

**注：**工程文件目录不能存在中文目录，否则综合工程的时候会报错。

### 1.2 工程的内容：

本工程实现了一个函数函数发生器的功能。关于函数发生器的介绍可以参考《Quartus学习要求安排》中的A题。

```
《Quartus学习要求》为14级学生的配套要求，之后的几届学生不再以此为要求，改成自由命题的形式。这里只是作为工程文件的配套教程使用。
```
